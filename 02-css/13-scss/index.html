Les bases de SASS et SCSS

  • Pour installer SASS on utilisera : npm install -g sass
  • La commande pour compiler du SASS est : sass nomFichier.sass nomFichier.css
  • SASS et SCSS ont exactement les même fonctionnalités. Seul la syntaxe change.
  • Pour éviter de retaper la commande à chaque fois, on peut lui indiquer de surveiller un fichier avec : sass nomFichier.scss nomFichier.css --watch
  • On peut aussi lui demander de surveiller tout un dossier avec: sass nomDossierSCSS/:nomDossierCSS --watch
  • Vous pouvez indiquer à votre compilateur de minimifier votre css avec : sass nomDossierSCSS/:nomDossierCSS --watch --style compressed
